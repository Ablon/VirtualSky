VirtualSky

Virtual Sky a browser-based planetarium from LCOGT that lets you see what is visible in the sky from any location on Earth. It can be customized and included on your own website, blog etc. It is provided freely for educational and non-profit use.

  1. A default view with a stereo projection. 
    $(document).ready(function() {

	var planetarium = $.virtualsky({
		id: 'starmap1',
		projection: 'stereo'
	});

});

    <div id="starmap1" style="width:100%;height:400px;"></div>
  2. A polar projection, with keyboard and mouse access set for Santa Barbara's lat/long: 
    $(document).ready(function() {

	var planetarium3 = $.virtualsky({
		id: 'starmap',
		projection: 'polar', 
		mouse: true, 
		keyboard: true, 
		latitude: 34.4326, 
		longitude: -119.86286
	});
});
    <div id="starmap" style="width:500px;height:500px;"></div>
  3. A lambert projection, with constellations shown, no keyboard control and a black-on-white view. It is based in Manchester,UK and facing north: 
    $(document).ready(function() {

	var planetarium = $.virtualsky({
		id: 'starmap', 
		projection: 'lambert', 
		mouse: true,
		keyboard: false, 
		az: 0, 
		negative: true, 
		constellations: true, 
		latitude: 53.5,
		longitude: -2.5
	});
});
    <div id="starmap" style="width:500px;height:300px;"></div>
  4. A stereo projection with no keyboard or mouse input and with a Galactic grid. The view is facing south from Hill Valley, California at 1:21 am on October 25, 1985. 
    $(document).ready(function() {

	var planetarium = $.virtualsky({
		id: 'starmap',
		projection: 'stereo',
		keyboard: false, 
		mouse: false, 
		az: 180, 
		latitude: 38.25, 
		longitude: -122.6, 
		clock: new Date("October 25, 1985 01:21:00"),
		gridlines_gal: true
	});
});
    <div id="starmap" style="width:650px;height:300px;"></div>
  5. A mollweide projection with with star labels drawn and the ground blocking half the sky. 
    $(document).ready(function() {

	var planetarium = $.virtualsky({
		id: 'starmap',
		projection: 'mollweide',
		showstarlabels: true,
		az: 180,
		ground: true
	});
});
    <div id="starmap" style="width:920px;height:460px;"></div>
